# User API

User API es un microservicio que simula un microservicio de usuarios. Tiene como objetivo buscar y retornar usuarios dentro del sistema.

## Requisitos

* [uv](https://docs.astral.sh/uv/)
* Python 3.11.x

## Inicializacion del servidor

Una vez que se clona el repositorio se debe realizar el `uv sync` para descargar los paquetes e inicializar la virtual env.

Para comenzar la applicacion, desde `user-service` se debe ejecutar el comando  `uv run python -m src.main`
